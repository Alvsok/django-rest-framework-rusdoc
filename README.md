# Руководство

Для запуска документации необходимо установить MkDocs

    sudo pip install mkdocs

Перейти в директорию с `mkdocs.yml`

Запустить встроенный веб-сервер MkDocs

    mkdocs serve

## Git

В проекте три ветки:
- master
- rus
- work

`master` - оригинал
`rus` - русская версия
`work` - рабочая версия с английским и русским вариантом
